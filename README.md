# La Gaceta de Cuauhtémoc

Boceto de un **directorio del comercio local de la Alcaldía Cuauhtémoc (CDMX)** con estética de periódico digital.

> ⚠️ Esto es un **boceto** con datos de muestra (no reales) para revisar dirección y diseño. No es la versión final.

## Idea

Un catálogo digital de los negocios del barrio, presentado como una gaceta. Doble propósito:
- **Recurso para la comunidad:** vecinos encuentran negocios; comerciantes ganan visibilidad gratis.
- **Herramienta de presencia territorial:** razón legítima para conocer a cada comercio y construir relación de cara a una candidatura.
